description: Set up or review an ML experiment. Spawns ml-engineer to design the experiment, write training code, and produce an evaluation framework.
---

Spawn `ml-engineer` with the ML task description (classification, regression, recommendation, etc.) and any available context.

Produces:
1. Problem framing and baseline definition
2. Feature engineering plan
3. Training script with MLflow/W&B tracking
4. Evaluation metrics (ML metrics + business metrics)
5. Model card template
