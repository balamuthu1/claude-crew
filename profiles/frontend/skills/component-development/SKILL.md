# Component Development Workflow

1. Spawn `frontend-architect` to design component API and state management
2. Spawn `frontend-developer` to implement the component
3. Spawn `accessibility-auditor` to verify WCAG 2.1 AA compliance
4. Spawn `ui-engineer` to review CSS and design token usage
5. Output: component implementation + tests + accessibility report
