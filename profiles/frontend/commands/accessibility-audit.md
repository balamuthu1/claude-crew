---
description: Full WCAG 2.1 AA accessibility audit for frontend components or pages. Spawns accessibility-auditor.
---

Spawn `accessibility-auditor` with the UI file paths.

Produces a full WCAG 2.1 AA audit covering:
1. Perceivable (alt text, contrast, captions)
2. Operable (keyboard, focus, skip nav)
3. Understandable (labels, errors, language)
4. Robust (valid HTML, ARIA correctness)

Output: structured report with Critical (WCAG fail) / Major / Minor findings and WCAG criterion references.
