# Product Discovery Workflow

1. Spawn `product-manager` to frame the problem and opportunity
2. Define the user pain point and current workarounds
3. Estimate impact (affected users, frequency, severity)
4. Explore 2-3 solution approaches with trade-offs
5. Recommend the option with best impact/effort ratio
6. Output: discovery document ready for stakeholder review
