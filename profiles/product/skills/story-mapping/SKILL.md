# Story Mapping Workflow

1. Spawn `user-story-writer` with the epic or feature description
2. Identify the user types and their goals
3. Map the user journey (walking skeleton)
4. Break the journey into stories (vertical slices)
5. Write acceptance criteria for each story
6. Sequence stories for MVP vs future releases
7. Output: story map + full story set with AC + sprint sequencing
