---
description: Write user stories for a feature or epic. Spawns user-story-writer to break down the feature into sprint-ready stories with acceptance criteria.
---

Spawn `user-story-writer` with the epic or feature description.

Produces:
1. Set of user stories in "As a... I want... so that..." format
2. Acceptance criteria in Given/When/Then format for each story
3. Edge cases and out-of-scope items
4. Story dependencies and sequencing
5. Definition of Ready checklist per story
