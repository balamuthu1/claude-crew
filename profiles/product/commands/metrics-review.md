---
description: Review or define product metrics. Spawns metrics-analyst to define KPIs, event schemas, or interpret metrics data.
---

Spawn `metrics-analyst` with:
- Feature or area to define metrics for (or existing metrics to review)
- Any available data context

Produces:
1. Primary metric with definition and calculation
2. Secondary metrics and guardrail metrics
3. Analytics event schema (what to track and how)
4. Baseline measurement plan
