description: Write a Product Requirements Document. Spawns prd-author to produce a complete, engineering-ready PRD.
---

Spawn `prd-author` with the feature description and any context provided.

Produces a complete PRD including:
1. Problem statement
2. Goals and non-goals
3. User stories
4. Functional requirements with edge cases and error states
5. Acceptance criteria
6. Success metrics
7. Dependencies and open questions
