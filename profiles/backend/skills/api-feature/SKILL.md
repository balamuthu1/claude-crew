# Build a New Backend API Feature

Follow the `/api-sdlc` pipeline to build the feature end-to-end:

1. **ARCHITECT** — design API contract, data model, service boundaries
2. **DEVELOP** — implement migration, model, repository, service, controller
3. **TEST** — write unit tests, integration tests, test data factories
4. **REVIEW** — code quality gate
5. **SECURITY** — OWASP API Security Top 10 audit
6. **DEPLOY CHECKLIST** — deployment readiness

See `/api-sdlc` for the full orchestration.
