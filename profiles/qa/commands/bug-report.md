---
description: Triage a bug and write a structured bug report. Spawns bug-triager to investigate, classify severity/priority, and produce a complete bug report.
---

Spawn `bug-triager` with the bug description and any available context (error messages, reproduction steps, affected version).

The triager produces:
1. Severity and priority classification
2. Structured bug report with reproduction steps
3. Root cause analysis (if identifiable from provided context)
4. Suggested fix approach
