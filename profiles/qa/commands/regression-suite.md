description: Generate or review a regression test suite. Spawns automation-engineer to write automated regression tests for critical paths.
---

Spawn `automation-engineer` with:
- The target feature or module
- Existing test framework from `qa.config.md`

The automation engineer identifies critical user journeys and produces automated regression tests covering:
1. Authentication flows
2. Core feature happy paths
3. Key error scenarios
4. Previously reported bugs (regression prevention)
