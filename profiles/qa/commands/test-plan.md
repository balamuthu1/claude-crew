---
description: Generate a test plan for a feature or release. Spawns test-strategist to produce a risk-based test plan with coverage strategy.
---

Spawn `test-strategist` with:
- Feature description or release scope
- Stack context from `qa.config.md` (if present)

Produce a test plan covering:
1. Scope and risk assessment
2. Test types required (unit, integration, E2E, performance)
3. Test cases (happy path, edge cases, negative cases)
4. Automation recommendations
5. Entry and exit criteria
