# Test Planning Workflow

1. Spawn `test-strategist` with the feature/release scope
2. Produce risk-based test plan with coverage strategy
3. Define test pyramid allocation for this feature
4. Generate test cases: happy path, edge cases, negative cases
5. Identify automation candidates vs manual test cases
6. Output: complete test plan ready for sprint
